# Plan de Dependencias del Proyecto Backend (pom.xml)

Este documento detalla las dependencias principales declaradas en el archivo [pom.xml](file:///c:/Users/Dell/Documents/Miercoles-Proyecto/backend/pom.xml) del proyecto y su respectivo propósito de forma simplificada:

| Dependencia | Propósito |
| :--- | :--- |
| **spring-boot-starter-web** | API REST con Tomcat embebido |
| **spring-boot-starter-security** | Framework de seguridad (Spring Security) |
| **spring-boot-starter-data-jpa** | Acceso a BD con Hibernate/JPA |
| **spring-boot-starter-validation** | Validación Bean Validation |
| **mysql-connector-j** | Conector de base de datos MySQL |
| **jjwt-api** / **jjwt-impl** / **jjwt-jackson** | Generación y validación JWT (v0.12.6) |
| **itextpdf** | Generación de reportes PDF |
| **lombok** | Reducción de código repetitivo (Getters, Setters, Builders) |
| **spring-boot-starter-test** | Framework de pruebas unitarias y de integración |
| **spring-security-test** | Soporte para pruebas de seguridad en endpoints |
